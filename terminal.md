#GNOME-TERMINAL
`ALT + R` new tab
`ALT + T` new window
`ALT + W` close window
`ALT + Q` close terminal

#ALIASES
`zshrc` nvim ~/.zshrc
`xsh` execute shell
`v` nvim
`ra` . ranger
`gclear` stash & stash clear
`mv` color-move
`cp` color-copypaste
`master` checkout master
`main` checkout main
`shortcuts` cd Shortcuts
`scripts` cd Scripts
`dotfiles` cd DotFiles
`vm` virt-manager
`books` books

#MOVEMENT
`CTRL + A` go to start
`CTRL + E` go to end
`CTRL + U` delete whole line
`CTRL + K` delete forward
`CTRL + L` clear
`CTRL + W` delete word 
`ALT + BS` delete word
